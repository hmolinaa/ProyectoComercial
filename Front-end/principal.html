Envio de calificaciones a estudiantes



Sobre lo que trata mi proyecto

Un profesor tiene una cuadro de calificaciones finales de sus estudiantes. El profesor desea enviar un mensaje de correo electronico personalizado a cada estudiante donde se detalle las calificaciones parciales y las calificaciones finales. Elabore un sistema web que cumpla con dicha tarea.

  • 1. Crear una lista de correos
  • 2. Componer el correo electrónico
  • 3. Personalizar el mensaje
  • 4. Enviar el correo electrónico
